# 戦車でゴルフ「Ball und Panzer Golf」 
 <img width="640" alt="title" src="../../Material/202306072325350729.png">

● 内容<BR>
　X68000で動くゲームです。一人プレイ専用です。<BR>
　戦車を操作してホールインワンを狙ってください。<BR>
　18ホールの全ての旗を破壊したらクリアです。<BR>
　ゴルフ同様、スコアを競って遊んでください。<BR>

● 操作方法<BR>
　アナログ操作（サイバースティック対応）<BR>
　　戦車の向き			：右レバー<BR>
　　戦車の移動			：左スロットル<BR>
　　ショット			：Aボタン（右レバーにあるボタン）<BR>
　　ボールを落下させる	：ショット後、Aボタン<BR>

　デジタル操作（ジョイカード、ジョイスティック、キーボード）<BR>
　　戦車の向き			：左、右<BR>
　　戦車の移動			：上、下<BR>
　　ショット			：Aボタン<BR>
　　ボールを落下させる	：ショット後、Aボタン<BR>

　キーボード操作<BR>
　　戦車の向き			：←、→、４、６<BR>
　　戦車の移動			：↑、↓、８、２<BR>
　　ショット			：Ｚ<BR>
　　ボールを落下させる	：ショット後、Ｚ<BR>

  入力切替				：TABキー<BR>
  終了					：ESCキー<BR>

● ルール<BR>
　ステージクリア条件：旗を全て破壊すること<BR>
　ゲームオーバー条件：持っているボールが全てなくなること<BR>

　「ショットするフェーズ」「移動するフェーズ」があります。<BR>　

　最初はゴルフ同様「ショットするフェーズ」です。<BR>
　戦車をいい感じの方向に回転した後、旗に目掛けてボールを「ショット」してください。<BR>
　一度「ショット」しただけでは、ボールは上昇していくだけなので<BR>
　適度なポイントで「ボールを落下させる」でボールを落としてください。<BR>
　上手く旗にボールを当てることができたら、旗を破壊することができます。<BR>
　また、「ショットするフェーズ」中は水色のエリア内だけ移動可能です。<BR>
　水色のエリア内に存在する旗をゲットすればボールが１つ増えます。<BR>

　旗を破壊したら「移動フェーズ」に変わります。<BR>
　戦車を操作して、破壊した旗から出てくる「エネルギー」を回収してください。<BR>
　「エネルギー」を回収すると「ショットするフェーズ」に変わります。<BR>

　色々なポイントでハイスコアにつながる要素があるので、発見しながら楽しんでください。<BR>
